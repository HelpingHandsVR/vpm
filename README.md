# Helping Hands VPM repository

This is a VPM repository that allows users creating VRChat content with [VCC (VRChat Creator Companion)](https://vcc.docs.vrchat.com/) or [ALCOM](https://vrc-get.anatawa12.com/en/alcom/) to easily pull in Helping Hands-related assets for worlds and avatars.

## Using the VPM repository

### VRChat Creator Companion

1. Open the VRChat Creator Companion
2. Click on Settings on the left side
3. Click on the Packages tab
4. Click "(+) Add Repository"
5. Paste `https://helpinghandsvr.github.io/vpm/index.json`.
6. Press "Add".

Now, from the Projects list, you can click "Manage Project" and add Helping Hands-related packages to any of your projects.

### ALCOM

1. Open ALCOM
2. Click on Resources on the left side
3. Ensure the Repositories tab is focused
4. Click "Add Repository"
5. Paste `https://helpinghandsvr.github.io/vpm/index.json`.
6. Press "Add Repository".

Now, from the Projects list, you can click "Manage" and add Helping Hands-related packages to any of your projects.
